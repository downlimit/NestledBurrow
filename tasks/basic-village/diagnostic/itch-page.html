Basic Village Tileset

A downloadable asset pack

Hey! 

This is a 16x16 tileset and collection of sprites for basic top-down village. 

Click to view full size (in new tab)



BasicVillageTileset archive contains:

  • Outroor_tileset.png - grass, dirt, river and stone mountains
  • House_tileset.png - tileset contains walls, fence, floor  and carpet
  • Trees_and_bushes.png - a small set of trees and bushes
  • Furniture.png - furniture for tavern, home and blacksmith's workshop


GodotTilemapImport:

has BOTH image and tilemap folders with created tilemaps with collision:

  • a_dirt_tileset.tres - autotiles for dirt
  • a_grass_tileset.tres - autotiles for grass 
  • a_water_tileset.tres - autotiles for water
  • s_stone_n_decoration_tiles.tres - single tiles for stone mountains, house walls, floor, fence, etc. 

I added Tileset_Merged.png for these tilemaps. but if you need - there are still House and Outdoor images separately.


License:
