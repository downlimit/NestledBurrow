# Временные исходные ассеты задачи pixel-room

Эти архивы автоматически загружены с официального сайта Kenney и нужны только для выполнения `../TASK.md`.

## Архивы

- `kenney_rpg-urban-pack.zip`
  - Источник: https://kenney.nl/assets/rpg-urban-pack
  - Автор: Kenney
  - Лицензия: CC0 1.0 Universal
  - Назначение: выбрать одного маленького человека с четырьмя направлениями ходьбы.

- `kenney_roguelike-rpg-pack.zip`
  - Источник: https://kenney.nl/assets/roguelike-rpg-pack
  - Автор: Kenney
  - Лицензия: CC0 1.0 Universal
  - Назначение: выбрать деревянный пол, четыре стороны стен и четыре угла комнаты.

Контрольные суммы находятся в `SHA256SUMS`.

После интеграции выбранных PNG весь каталог `tasks/pixel-room/`, включая эти полные архивы, должен быть удалён из итоговой ветки Codex.
