# NestledBurrow: текущий вектор

## Где находится проект

Создан минимальный браузерный проект на Phaser 4.2.1 и Vite. Он собирается и показывает стартовую сцену. Репозитория GitHub и публичной игровой ссылки пока нет.

## Текущая цель

Наладить минимальный рабочий путь:

`задача в ChatGPT → изменение кода → проверка → GitHub → запуск игры по ссылке`

## Следующий шаг

1. Создать пустой репозиторий GitHub и дать подключённому плагину доступ к нему.
2. Перенести туда существующий стартовый проект.
3. Настроить автоматическую публикацию браузерной сборки.
4. После этого сделать первую техническую игровую проверку: пустая комната и временный объект игрока с движением.

## Текущий блокер

GitHub-плагин подключён, но пока не видит ни одного доступного репозитория.

## Действующие решения

- Основное общение и постановка задач происходят в ChatGPT с телефона или компьютера.
- Игра создаётся как браузерная 2D-игра на Phaser 4.
- Код и проектные документы должны храниться в GitHub.
- Основной чат управляет направлением работы; кодовый агент выполняет ограниченные задачи, после чего результат проверяется.
- Проектные знания разделяются на тематические «здания» только по фактической необходимости; список зданий заранее не придумывается.
