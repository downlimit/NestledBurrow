# NestledBurrow: текущий вектор

## Где находится проект

Создан минимальный браузерный проект на Phaser 4.2.1 и Vite. Он собирается и показывает первую техническую комнату с временным игроком. Репозиторий `downlimit/NestledBurrow` подключён; изменения проходят через отдельные ветки и pull request. GitHub Pages работает по адресу: https://downlimit.github.io/NestledBurrow/.

## Текущая цель

Наладить минимальный рабочий путь:

`ChatGPT формулирует задачу → Codex делает изменения → пользователь нажимает «Создать PR» → ChatGPT проверяет PR → после подтверждения выполняется merge`

## Следующий шаг

Следующий шаг должен быть определён в основном чате после завершения текущей публикационной проверки; следующий игровой шаг заранее не придумывается.

## Текущий блокер

Текущих инфраструктурных блокеров нет.

## Действующие решения

- Основное общение и постановка задач происходят в ChatGPT с телефона или компьютера.
- Игра создаётся как браузерная 2D-игра на Phaser 4.
- Код и проектные документы должны храниться в GitHub.
- GitHub Pages работает по адресу: https://downlimit.github.io/NestledBurrow/.
- Workflow проверки pull request добавлен.
- Первая техническая проверка реализована: пустая комната, визуальные границы и временный прямоугольник игрока с движением.
- Мобильный виртуальный джойстик реализован для touch/coarse pointer устройств; клавиатурное управление сохранено.
- Опубликованная версия определяется через `/version.json`, где хранится полный SHA опубликованного коммита и его короткий идентификатор.
- Основной чат считает эту задачу завершённой только после успешного PR check, merge и совпадения `/version.json` с merge-коммитом.
- Рабочий цикл: ChatGPT формулирует задачу → Codex делает изменения → пользователь нажимает «Создать PR» → ChatGPT проверяет PR → после подтверждения выполняется merge.
- Основной чат управляет направлением работы; кодовый агент выполняет ограниченные задачи, после чего результат проверяется.
- Проектные знания разделяются на тематические «здания» только по фактической необходимости; список зданий заранее не придумывается.
